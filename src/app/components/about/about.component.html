Sobre Super Héroes App

Esta es una aplicación realizada con Angular CLI versión 11.2.0 con fines de estudio y para practicar el desarrollo Frontend con Angular y Bootstrap.
Se pueden hacer búsquedas de super heroes en la barra de navegación.
Los resultados de la búsqueda se traen desde un servicio leyendo los objetos json de un archivo.
Las imágenes que usé cuentan con licencias Creative Commons (CC BY'NC 4.0)


Tecnologías utilizadas en esta aplicación

  • Angular CLI 11
  • Componentes reutilzables, Servicios y Rutas
  • Directivas estructurales (*ngFor y *ngIf)
  • Enviar datos del componente padre al hijo a traves de @Input y [prop],comunicar al componente hijo con el padre a través de @Output y EventEmmiter
  • Transformación visual de los datos usando pipes


  • Bootstrap 4.6
  • NavBar
  • Cards
  • Jumbotron


  • Otras tecnologías
  • HTML 5
  • CSS 3 (Cambios estéticos y uso de la libreria 'animate')
  • Typescript
